ToGO

The full-stack framework that ships your Go backend and React frontend as a single deployable app — a Laravel-artisan-like CLI for the Go + sqlc + Atlas + React stack.

$ curl -fsSL https://to-go.dev/install.sh | sh
Get started → npm i -g @togo-framework/cli

One repo, one binary

Your Go API and React UI compile into a single artifact. No glue, no orchestration.

artisan-grade CLI

togo new, make:resource, migrate, generate, serve — generators first.

Everything is a plugin

A thin microkernel. Auth, dashboard, DB drivers, cache, queue — installed via togo install.

Pick your database

sqlite by default · postgres · togo-postgres (ParadeDB + pgvector + pg_cron) · supabase · mysql · mongodb.

$ npm install -g @togo-framework/cli
$ togo new myapp        # pick frontend (TanStack/Next.js) + database
$ cd myapp && togo serve
→ serving on http://localhost:8080  # Go API + React UI, one binary
