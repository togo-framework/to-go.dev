<h1 align="center">to-go.dev</h1>
<p align="center">The togo framework website — landing, docs, and the install.sh/update.sh scripts.</p>

Static site served by Caddy on the togo box. `public/` is the web root; `Caddyfile` is the server config; `deploy.sh` pulls + reloads. `curl -fsSL https://to-go.dev/install.sh | sh` installs the CLI.
