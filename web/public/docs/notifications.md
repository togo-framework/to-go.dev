<h1 align="center">notifications</h1>
<p align="center"><sub>part of the <a href="https://github.com/togo-framework">togo-framework</a> — the full-stack Go + React framework</sub></p>

Laravel-style notifications for [togo](https://github.com/togo-framework/togo).
Channels: **mail** (via togo mail), **broadcast** (via togo realtime), **database**.
Push providers (FCM, OneSignal, Pusher) ship as channel plugins.

```bash
togo install togo-framework/notifications
```
