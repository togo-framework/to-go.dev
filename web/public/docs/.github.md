# .github

Organization-wide community health files for [togo-framework](https://github.com/togo-framework).

These defaults apply to **every** repo in the org that doesn't define its own:

- `.github/ISSUE_TEMPLATE/` — bug report + feature request forms
- `.github/PULL_REQUEST_TEMPLATE.md` — PR checklist
- `.github/FUNDING.yml` — Sponsor button ([@fadymondy](https://github.com/sponsors/fadymondy))
- `profile/README.md` — the org landing page
