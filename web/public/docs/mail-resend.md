<h1 align="center">mail-resend</h1>
<p align="center"><sub>part of the <a href="https://github.com/togo-framework">togo-framework</a> — the full-stack Go + React framework</sub></p>

[Resend](https://resend.com) driver for togo [mail](https://github.com/togo-framework/mail).
Demonstrates the driver-plugin pattern (plugins depending on plugins).

```bash
togo install togo-framework/mail
togo install togo-framework/mail-resend
```

Set `MAIL_DRIVER=resend` and `RESEND_API_KEY`.
